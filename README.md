# Tic-Tac-Toe Game Website

A simple tic-tac-toe game built with Flask.

## Installation

1. Install Python 3.x
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open http://127.0.0.1:5000/ in your browser

## How to Play

- Click on empty cells to place your mark (X or O)
- First to get three in a row wins
- Click "Reset Game" to start over

## Troubleshooting

- If the app doesn't start, ensure Flask is installed
- For production, set a proper secret key in app.py
